4채널 아키텍처

4개의 전용 채널로 모든 이해관계자를 연결합니다.

DISPATCH — 관리자 통합 대시보드

예약·배차·관제·정산을 하나의 웹 화면에서 관리합니다.

  • 예약 접수 → 배차 → 운행 → 정산 원스톱
  • 실시간 GPS 관제 (5초 갱신)
  • AI 자동 견적 (20개+ 변수, 95%+)
  • 매출·운행 데이터 대시보드
디스패치 대시보드 화면

CONCIERGE — 거래처 B2B 전용 예약

호텔·여행사·기업 고객이 직접 예약하고, 운행 현황을 실시간으로 확인합니다.

  • 거래처 전용 예약 포탈
  • 운행 현황 실시간 공유
  • 월별 자동 정산 리포트
컨시어지 앱 화면

PARTNERS — 기사 전용앱

기사 전용앱으로 배차확인, 경로안내, 운행 스케줄을 관리합니다.

  • 파트너사 배차요청 · 수락
  • 차량·기사 풀 공유
  • 수행일지 작성
파트너스 화면

BOOKING — 해외고객 D2C 예약

다국어 예약 페이지와 글로벌 결제로 해외 고객을 직접 유치합니다.

  • 다국어 예약 페이지 (EN/CN/JP/TH)
  • 글로벌 결제 8종 (PayPal·VISA·MC 등)
  • 자사 브랜드 D2C 채널 구축

2모드 배차 방식

배차 요청형

기사가 직접 요청을 수락하는 온디맨드 방식으로 소규모 운영에 적합합니다.

  • 배차 요청 → 기사 수락 → 배정 확정
  • 소규모 관광대절 · 택시 최적
  • 기사 부하 자동 분산

배차 확정형

관리자가 직접 기사를 지정하는 방식으로 VIP 의전·대규모 운영에 적합합니다.

  • 관리자 직접 배차 지정
  • VIP 의전 전용 워크플로우
  • APEC 수송포탈 실적 검증
